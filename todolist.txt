1. Выбирать текст из wr -> wt
2. Учитывать все кейсы, показанные на картинке
3. Формировать логические блоки, учитывая вложенность тегов
4. Отправлять полученные логические блоки на бек и затем на фронт


Структура:
{"w:body":
	{"w:p":
		{
		{},
		{},
		{"":"",
		"w:r":{}
		}
		}
	}
}

Количество объектов внутри массива - кол-во строк (1 абзац на N строк всё равно считается как 1)
